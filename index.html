보고서 추가 방법
1. reports/ 폴더에 HTML 파일을 넣는다.
2. reports.json에 항목을 추가한다:
{
  "id": "report-05",
  "title": "보고서 제목",
  "file": "reports/report-05.html",
  "category": "관측 보고서",
  "tags": ["태그1", "태그2"],
  "date": "2025-06-01",
  "description": "간략한 설명",
  "starred": false
}
3. GitHub에 커밋하면 자동으로 반영된다.
